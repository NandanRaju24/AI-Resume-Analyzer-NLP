## Demo
# AI-Resume-Analyzer-NLP
Run the app:
streamlit run app.py

AI-Powered Resume Analyzer using NLP

Built an end-to-end NLP system to match resumes with job descriptions using TF-IDF and cosine similarity
Implemented skill extraction and weighted scoring to improve matching accuracy and relevance
Developed an interactive Streamlit web application for real-time resume analysis
Designed an explainable system highlighting matching and missing skills for better decision-making
